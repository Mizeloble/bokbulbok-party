# 성장·수익화 전략 (2026-07 기준)

> 전제: 코드·인프라는 완성 상태(광고/분석 env 게이팅, scale-to-zero로 유휴 비용 ~0). 병목은 **트래픽**.
> 실측(2026-07-19): 최근 30일 방 생성 1개. 즉 지금은 0단계.
> 운영 절차는 [launch-checklist.md](launch-checklist.md), 이 문서는 "언제·무엇을·왜"의 전략.

---

## 0. 현실적인 돈 계산 (기대치 고정)

배너 광고(애드핏 320×50, 대기화면 3곳) 기준 대략:

| 하루 방문 | 월 방문 | 예상 월 광고수익 | 의미 |
|---|---|---|---|
| ~30 | ~1천 | 몇백 원 | 서버비도 안 됨 |
| ~300 | ~1만 | 1~5만 원 | 서버비 충당선 |
| ~3,000 | ~10만 | 10~50만 원 | 용돈 |
| ~30,000 | ~100만 | 100만 원+ | 사업 검토선 |

- eCPM(1천 노출당 수익)은 애드핏 수백 원, 애드센스 1천~3천 원 수준에서 변동. 정확한 값은 켜봐야 안다.
- **교훈: 광고는 트래픽의 함수. 광고 세팅을 서두를 이유가 전혀 없고, 모든 노력은 트래픽에 투입.**

## 1. 단계 정의와 승격 기준 (Prometheus 수치로 판단)

측정 명령은 launch-checklist E절. 판단은 CF 페이지뷰가 아니라 **방 생성 수**로.

- **Stage 0 → 1 (검증)**: 내가 뿌리지 않은 주에 방 생성 > 0 이 나오면 통과.
- **Stage 1 → 2 (성장)**: 주당 방 30개+, 4주 연속 → 도메인 구입 + 애드핏 심사 착수.
- **Stage 2 → 3 (확장)**: 주당 방 300개+ → VM 수직 확장, MAX_ROOMS 상향, 애드센스 병행 검토.
- **킬 기준**: 6개월(=2027-01) 동안 Stage 1 미달이고 본인도 재미가 식었으면, 광고 계획은 접고 포트폴리오·오픈소스 자산으로 전환. (서버비 ~0이라 방치 유지도 무방.)

## 2. Stage 0: 제품 검증 (지금 ~ 수 주)

돈 쓰는 마케팅 전에, "모르는 사람이 재밌어하는가"부터.

- [ ] **실전 투입 3회+**: 실제 술자리·모임에서 직접 호스트로 사용. 관찰 포인트: QR 입장에서 막히는 사람, 게임 이해 못 하는 순간, "링크 뭐야?" 반응 여부.
- [ ] **자발적 재사용 신호**: 내가 없는 자리에서 누가 다시 열었는가(= 뿌리지 않은 방 생성). 이게 Stage 1 통과 신호.
- [ ] 피드백 창구: `FEEDBACK_URL`(구글폼) 등록 — variable 하나로 푸터 노출.
- 실패 신호가 나오면: 마케팅이 아니라 **온보딩/게임성 수정**이 먼저. (전환 깔때기: 랜딩 도착 → 방 생성 → 2명+ 입장 → 게임 시작 → 재방문. 어디서 새는지.)

## 3. Stage 1: 트래픽 만들기 (핵심 챕터)

웹 파티게임의 유입은 ①검색 ②커뮤니티 ③숏폼 ④내장 바이럴 4개. 순서대로 비용↑ 효과↑.

### 3-1. 검색 (기본값, 자는 동안 일함)
- **구글**: 랜딩 SEO — "술게임", "벌칙 정하기", "복불복 게임", "MT 게임", "회식 게임" 키워드. 게임별 소개 페이지 + FAQ/구조화 데이터. *(코드 작업 가능)*
- **네이버 (한국 검색의 절반)**: 구글 SEO로는 안 잡힘. **네이버 블로그 후기 글**이 실질 유입원 — 직접 1~2개 작성("술자리에서 폰으로 하는 복불복 게임 후기" 톤) + 서치어드바이저 등록.

### 3-2. 커뮤니티 시딩 (저비용, 반복)
- 타깃: 대학생(에브리타임·캠퍼스픽), 일반(디시 주류/게임 갤, 더쿠, 네이트판), 직장인(블라인드 — 회식 컨텍스트).
- **원칙: 광고 티 내면 죽는다.** "이거 만들어봤는데 피드백 부탁" (개발자 톤, 디시·클리앙·긱뉴스 등) 또는 실사용 후기 톤. 한 곳에 붙으면 그 채널 반복.
- 오픈소스 각: GitHub 트렌딩·긱뉴스(news.hada.io)·velog 개발기 — 개발자 트래픽은 광고 수익은 낮지만 초기 피드백·전파력이 좋음.

### 3-3. 숏폼 (가장 확률 높은 대박 경로)
- 술게임·파티게임은 릴스/쇼츠/틱톡과 상성이 최고. **실제 플레이 리액션 15~30초** 영상(마블 레이스 결과에 환호/절규하는 순간) + "링크는 프로필에".
- 모임 있을 때마다 촬영해 두는 것만으로 소재 확보. 계정 하나 파서 주 1~2개, 3개월.
- 인플루언서 협업은 Stage 2에서 — 지금은 오가닉으로 반응 테스트.

### 3-4. 내장 바이럴 루프 *(코드 작업 가능)*
- 이미 있음: QR 입장, 카카오톡 OG 카드, 결과 공유 카드(`share-card.ts`), 결과 화면 "친구 부르기" 재초대(v2.12.0). 게임 특성상 **1 host → N 참가자** 구조 자체가 바이럴.
- SEO도 반영됨(v2.12.0): 키워드 메타·JSON-LD(WebApplication+FAQ)·랜딩 FAQ·게임별 소개 페이지(`/games/<id>`)·sitemap. 남은 건 네이버 블로그·서치어드바이저(사용자 액션).

### 3-5. 시즌 캘린더 (트래픽은 계절을 탐)
- **9월 초 개강·MT 시즌이 최대 기회** → 8월에 SEO·시딩·숏폼 준비 완료가 목표.
- 12월 연말 회식/홈파티 → 2차 피크. 3월 신학기 → 3차.
- 여름(지금)은 준비 기간으로 적기.

## 4. 수익화 스택 (트래픽 단계별)

1. **후원 (지금부터 가능, 트래픽 무관)**: 토스 아이디 송금 링크 or Buy Me a Coffee를 푸터/결과 화면에. 기대수익 낮지만 비용 0, 서비스 톤 해치지 않음.
2. **애드핏 배너 (Stage 2)**: launch-checklist 그대로. 도메인 선행 필수(*.fly.dev 심사 리스크).
3. **애드센스 병행 (Stage 3)**: eCPM 높지만 심사·동의 이슈. 트래픽 검증 후.
4. **선택지 (Stage 3+, 신호 있을 때만)**:
   - 커스텀 벌칙팩/테마 유료화 (B2C 마이크로 결제)
   - 행사·워크숍용 아이스브레이커 B2B (기업 레크리에이션·MT 대행사) — 방향 전환급이라 별도 결정
   - 절대 금지 유지: 게임 진행 화면 광고 (CLAUDE.md 원칙)

## 5. 비용 가드레일

- 현 구조 유휴 비용 ~0 (scale-to-zero). **고정비를 만들지 말 것** — 유료 분석·유료 마케팅 툴 금지.
- Stage 1 통과 전 지출 상한: 0원. Stage 2 진입 시: 도메인 연 1~2만 원이 첫 지출.
- 유료 광고 집행(인스타 부스트 등)은 오가닉으로 반응 확인된 크리에이티브가 있을 때만, 월 상한 정해서.

## 6. 루틴

- **주 1회**: Prometheus 방 생성 수 확인 → 단계 기준과 대조 (launch-checklist E절 명령).
- **모임 있을 때마다**: 실전 투입 + 숏폼 소재 촬영.
- **월 1회**: 채널별 회고 — 안 붙는 채널은 접고 붙는 채널에 집중.
